MiniUI – IT-Park Test Project
MiniUI — bu IT-Park testi doirasida yaratilgan zamonaviy va foydalanuvchi uchun qulay interfeys komponentlari to'plami. Ushbu loyiha o'z ichiga kreativ dizayn va funksionallikni oladi.

🚀 Loyiha haqida
Ushbu loyiha IT-Park kurslari davomida olingan bilimlar asosida yaratildi. Asosiy maqsad — foydalanuvchilar uchun sodda, ammo samarali interfeys elementlarini (UI kit) ishlab chiqish va amaliyotda qo'llashdir.(Bu oddiy ui bo'lgan bo'lsa ham..)

✨ Xususiyatlari:
Modern Design: Zamonaviy va toza vizual ko'rinish.

Responsive: Barcha qurilmalarda (mobil, planshet, desktop) to'g'ri ko'rinish.

Lightweight: Juda tez yuklanuvchi va optimallashgan kod strukturasi.

IT-Park Standard: O'quv standartlari va eng so'nggi texnologiyalar asosida yozilgan.

🛠 Texnologiyalar
Loyihani yaratishda quyidagi vositalardan foydalanilgan:

HTML5 & CSS3 (Flexbox, Grid)

JavaScript (Interaktiv elementlar uchun)

GitHub (Versiyalarni boshqarish uchun)
