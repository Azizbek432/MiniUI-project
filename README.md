# MiniUI – IT-Park Test Project

**MiniUI** — bu IT-Park testi doirasida yaratilgan zamonaviy va foydalanuvchi uchun qulay interfeys komponentlari (UI Kit) to'plami. Ushbu loyiha o'z ichiga kreativ dizayn va yuqori funksionallikni oladi.

---

## 🇺🇿 Loyiha haqida
Ushbu loyiha **IT-Park** kurslari davomida olingan bilimlar asosida yaratildi. Asosiy maqsad — foydalanuvchilar uchun sodda, ammo samarali interfeys elementlarini ishlab chiqish va amaliyotda qo'llashdir. Garchi bu oddiy UI to'plami bo'lsa-da, u real loyihalar uchun poydevor bo'la oladi.

### ✨ Xususiyatlari:
* **Modern Design:** Zamonaviy va toza vizual ko'rinish.
* **Responsive:** Barcha qurilmalarda (mobil, planshet, desktop) to'g'ri ko'rinish.
* **Lightweight:** Juda tez yuklanuvchi va optimallashgan kod strukturasi.
* **IT-Park Standard:** O'quv standartlari va zamonaviy texnologiyalar asosida yozilgan.

---

## 🇬🇧 About the Project
**MiniUI** is a collection of modern and user-friendly interface components developed as part of an **IT-Park** test. The project focuses on creative design and practical functionality.

The primary goal was to develop a simple yet effective UI kit based on the knowledge gained during IT-Park courses. Even as a fundamental UI project, it demonstrates core frontend principles.

### ✨ Key Features:
* **Modern Design:** Clean and contemporary visual appearance.
* **Responsive:** Fully functional across all devices (mobile, tablet, desktop).
* **Lightweight:** Optimized code structure for fast loading speeds.
* **IT-Park Standard:** Built following educational standards and modern web technologies.

---

## 🛠 Texnologiyalar | Technologies
Loyihani yaratishda quyidagi vositalardan foydalanilgan:

* **HTML5 & CSS3** (Flexbox, Grid) — Struktura va moslanuvchan dizayn uchun.
* **JavaScript** — Interaktiv elementlar va dinamika uchun.
* **GitHub** — Versiyalarni boshqarish va deployment uchun.
